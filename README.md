Pokémon Explorer (Java)

A web application built using Java Servlets that fetches and displays real-time Pokémon data from the PokéAPI. This project demonstrates REST API integration, HTTP requests, JSON data processing, and dynamic content rendering.

🚀 Features

- Fetches random Pokémon data
- Displays Pokémon name and sprite
- Real-time API integration using PokéAPI
- Clean and simple user interface
- Fast and responsive

🛠️ Tech Stack

- Java
- Java Servlets
- HTML5
- CSS3
- JavaScript
- PokéAPI
